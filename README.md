# 💊 PharmaFlow  
### Smart Medical Store Management System

PharmaFlow is a modern, offline-first medical shop management system designed to streamline pharmacy operations such as inventory tracking, billing, and supplier management. It is built with a focus on simplicity, performance, and real-world usability.

---

## 🚀 Features

- 🧾 **Billing System**
  - Fast and accurate invoice generation
  - Automatic total calculation

- 💊 **Medicine Management**
  - Add, update, delete medicines
  - Track expiry dates and stock levels

- 📦 **Inventory Control**
  - Real-time stock updates
  - Low stock alerts

- 🧑‍💼 **Supplier Management**
  - Maintain supplier records
  - Track purchase history

- 🔍 **Search & Filter**
  - Quickly find medicines and records

- 💻 **Offline Support**
  - Works without internet (desktop-focused)

## ⚙️ Installation & Setup

### 1. Clone the repository
```bash
git clone https://github.com/codjav/PharmaFlow.git
cd PharmaFlow

2. Install dependencies

Client
cd client
npm install

Backend
cd ../backend
npm install

3. Setup Environment Variables

Create a .env file in the server folder:
PORT=5001
DATABASE_URL=your_database_connection

4. Run the project

Start Backend
cd server
npm run dev

Start Frontend
cd client
npm start
